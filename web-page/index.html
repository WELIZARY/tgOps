Управляй инфраструктурой прямо из Telegram

tgOps – Telegram-бот, который превращает чат в полноценную консоль DevOps-инженера: мониторинг серверов, управление Docker, деплой через CI/CD, бэкапы и алерты – в один клик.

Когда сервер не ждёт
а ноутбук далеко

Идея простая: продакшн падает в выходные, ноут не всегда под рукой, а Telegram – всегда. tgOps закрывает рутинные операции DevOps-инженера прямо из мессенджера: проверить состояние сервера, перезапустить контейнер, выкатить релиз или откатить пайплайн – за несколько секунд из любой точки мира.

Мониторинг в реальном времени

CPU, RAM, диск, сетевой I/O и здоровье контейнеров – обновляются прямо в чате с inline-кнопками детализации.

Алерты в Telegram

Любой webhook прилетает в чат с уровнем severity и кнопкой «Acknowledge» – критичные инциденты не потеряются.

Управление контейнерами

start, stop, restart, logs для Docker – без SSH-сессий и копипасты в терминал.

Деплой одной командой

Триггер пайплайна GitHub Actions / Jenkins из чата – с подтверждением и live-логом сборки.

Безопасный доступ

Whitelist Telegram-ID, RBAC по командам, аудит-лог в PostgreSQL – кто, когда и что запустил.

  • RBAC
  • Audit log
  • 2FA via TG
  • Rate limit

Бэкапы по расписанию

cron + S3-совместимое хранилище. Восстановление в один клик с превью размера и даты.

SSL / сертификаты

Напомнит за 14 дней до истечения и предложит обновить через Certbot – прямо из диалога.
